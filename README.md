# SimpleRSS
 A Simple RSS Reader

 ## How to Install
 - Download to a folder
 - Enable Chrome Extensions developer mode
 - Load unpacked and select the SimpleRSS folder

## How to Use
 - Copy feed/xml link into the text box and push 'Add Feed' button
 - When feed has unread entries, a 'New' icon will appear
 - Push 'Mark Read' button to update the feed's "Last Updated" time
 - Delete single feed with 'Remove' button
 - Delete all feeds with 'Clear Feeds' button (requires confirmation)
